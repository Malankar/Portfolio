## Getting Started

First, Install all the packages:

```bash
npm install  
#or 
yarn
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Packages:

```
React
Nextjs
React Toastify
React Icons
EmailJS (for Contact Us)
```
